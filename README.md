# Zomato_analysis
Analysis of Zomato dataset to uncover restaurant insights and trends.

🍽️ Zomato Data Analysis & Dashboard

📌 Project Overview

This project focuses on analyzing Zomato food delivery data to understand **customer ordering behavior, restaurant performance, and sales trends**. The goal is to generate actionable insights to improve business strategies and customer experience. 

---

🎯 Objectives

* Analyze Zomato food delivery performance
* Understand customer ordering behavior
* Identify top-performing restaurants and cities
* Improve sales, customer retention, and operations 

---

📊 Dataset Overview

The dataset includes:

* Restaurant details (Name, City, Location)
* Cuisines and categories
* Ratings and votes
* Pricing and cost for two
* Online delivery & table booking availability
* Revenue and order trends

---

🧹 Data Preparation

* Cleaned dataset using Excel
* Removed duplicates and handled missing values
* Standardized columns for analysis
* Structured data for SQL queries

---

🗄️ Database (SQL)

* Created database and table structure
* Inserted restaurant data
* Performed queries for insights such as:

  * Top restaurants
  * Ratings analysis
  * City-wise performance
  * Online delivery trends 

---

📊 Tools Used

* Excel
* SQL
* Tableau
* Power BI 

---

📈 Key Insights

* Metro cities generate the highest revenue
* Top restaurants contribute major sales share
* Weekend orders are higher than weekdays
* Higher ratings increase order frequency
* Certain cuisines dominate customer preference 

---

⚠️ Low Performance Areas

* Some cities and restaurants show low revenue and orders

Improvements:

* Offer discounts during low demand
* Target city-specific marketing
* Encourage customer reviews
* Promote low-performing restaurants 

---

📊 Dashboard

Interactive dashboards created using:

* Power BI
* Tableau

Includes:

* KPIs (Average Rating, Total Restaurants, Revenue)
* Filters (City, Cuisine, Year, Price Range)
* Trend analysis and comparisons
